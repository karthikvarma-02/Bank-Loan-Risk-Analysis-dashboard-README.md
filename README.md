# Bank Loan Performance, Risk & Profitability Analysis (Power BI)

## Project Objective
The objective of this project is to analyze the bank’s loan portfolio using historical loan data to:
• Measure overall loan performance using key financial KPIs
• Identify risk-heavy segments contributing to loan defaults and revenue loss
• Track funded vs received amounts to understand repayment efficiency
• Support data-driven decisions that help increase profit and reduce charge-offs

## Tools Used
Power BI Desktop  
Power Query Editor  
DAX Measures  
Data Modeling

## Dataset
The dataset contains 38,576 loan records including:
• Loan amount
• Interest rate
• Installment
• Debt-to-Income ratio
• Annual income
• Employment length
• Loan grade
• Loan purpose
• Loan status
• Payment history

## Key Metrics
Total Loan Applications – 38.58K  
Total Funded Amount – 435.76M  
Total Amount Received – 473.07M  
Average Interest Rate – 12.05%  
Good Loan Percentage – 86.18%  
Bad Loan Percentage – 13.82%

## Outcome
An interactive Power BI dashboard was built to analyze loan performance, identify high-risk segments, and support data-driven lending decisions.
